UTF-8 Unicode Composer

# Synopsis

    ./utf8_unicode_composer INTEGER UTF8_STRING

# Example

Display all composed variants for `o`

    ./utf8_unicode_composer 64 o

# Build

1. Clone the project.
2. Change into the project directory.
3. Type `make`
